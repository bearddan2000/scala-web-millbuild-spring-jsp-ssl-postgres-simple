# scala-web-millbuild-spring-jsp-ssl-postgres-simple

## Description
A jsp springboot scala gradle build,
that connects to self-signed ssl enabled postgres database.

## Tech stack
- springboot
  - web
- jsp
- millbuild
  - 6.8.2
- bootstrap
- jquery
- dataTable

## Docker stack
- alpine
- postgres
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
